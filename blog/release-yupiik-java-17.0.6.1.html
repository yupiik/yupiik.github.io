    Yupiik Java 17.0.6.1 Docker Image Released

    , Francois Papon, 2023-03-17, 30 sec read

    On behalf the Yupiik OSS team and other contributors, I’m happy to announce that Yupiik Java 17.0.6.1 Docker Image has been released. The artifacts are available on Docker Hub.

    What’s new?

    The docker image is based on Zulu Java 17.0.6 JDK

    How to use?

    You can install the image in your local docker registry:

    docker pull ossyupiik/java:latest

    Feedback are welcome

    You can contribute, report bug or ask for improvement in the GitHub issue page

    Enjoy!
